Project Information

Project Name: Personal Portfolio Website

Date: January 2024

Overview: A fully responsive website showcasing my skills, experience, and achievements.

Features:

  • Multi-page navigation (Resume, Projects, Education, Contact).
  • Custom CSS styling and responsive design.

Technologies Used: HTML, CSS, JavaScript

Outcome: A professional online presence that highlights my technical skills.

Description

This website is a long process of building that has taken me longer than your average web developer. I wanted to be able to share more details about my experience and life with employers or anyone interested. I designed this website to be easy to read and make it simple to contact me! This project taught me more about front-end design than I ever knew. I find learning a new skill like HTML is best done through a project like this.
